# エラーコード一覧

## エラーコードの種別
| Code | Description |
|------|-------------|
| 10xx | 共通エラー    |
| 20xx | 認証エラー    |
| 30xx | APIエラー    |

## エラーコード内容
| Code | Description |
|------|-------------|
| 3001 | 存在しない注文番号です    |
| 3002 | 既に受領された注文番号です |
| 3003 | 既に出荷された注文番号です |
| 3004 | 存在しないjancodeです    |
| 3005 | 存在しないsku_codeです   |
